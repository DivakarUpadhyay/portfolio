## Executive Summary

a16z crypto submitted formal responses to the SEC's Crypto Task Force regarding tokenized securities and broker-dealer regulations. The firm advocates for modernizing U.S. securities laws to accommodate blockchain technology while preserving investor protections.

Tokenizing assets—stocks, bonds, funds—could enable:
- **24/7 markets** - No more trading hours
- **Faster settlement** - T+0 instead of T+2
- **Improved liquidity** - Fractional ownership
- **Broader access** - Lower minimum investments

## Recommendation 2: Modernize Transfer Agent Rules

### The Problem

Current regulations assume **"paper-era" infrastructure**.

### The Request

> The SEC should clarify that immutable onchain records can satisfy existing recordkeeping requirements and that "offchain records shouldn't be presumptively required."

| Current Rule | Proposed Update |
|--------------|-----------------|
| Paper-based records required | Onchain records acceptable |
| Centralized custody | Distributed ledger compatible |
| Manual reconciliation | Automated verification |

## Recommendation 4: Prevent State-Level Obstruction

### The Request

Federal preemption should apply to tokenized securities classified as **"covered securities"**, preventing conflicting state regulations.

```
REGULATORY CONFLICT EXAMPLE
════════════════════════════════════════════════════════════

  FEDERAL LAW                    STATE LAW (e.g., NY BitLicense)
  ───────────                    ────────────────────────────────

  Securities compliance          Additional requirements
  Investor protections           Separate licensing
  National standards             Conflicting rules

  Result: Innovation inhibited, compliance costs increase

════════════════════════════════════════════════════════════
```

## Recommendation 5: Enable Atomic Settlement

### Benefits

| Feature | Traditional | Atomic Settlement |
|---------|-------------|-------------------|
| Settlement Time | T+2 | T+0 (instant) |
| Counterparty Risk | Present | Eliminated |
| Failed Trades | Common | Impossible |
| Reconciliation | Required | Automatic |

### Request

> The SEC should provide guidance on integrating atomic settlement with existing market infrastructure.

---

## Recommendation 6: Avoid Over-Regulating Tokenized Trading

Most existing trading rules already support tokenized securities.

### Recommended Approach

| Action | Rationale |
|--------|-----------|
| Provide exemptive relief | ATSs handling certain transactions |
| Maintain existing rules | Where already compatible |
| Technology-neutral stance | Focus on outcomes, not methods |
