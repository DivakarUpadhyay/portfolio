## Note 1: Trading as a Waypoint, Not Destination

**Author:** Arianna Simpson, a16z crypto General Partner

![Trading Platforms](https://images.unsplash.com/photo-1611974789855-9c2a0a7236a3?w=1000&h=400&fit=crop)

### The Observation

Many successful crypto companies have pivoted toward trading platforms, creating market saturation.

> "Having so many players all doing the same thing cannibalizes mindshare for the many."

### The Marshmallow Test

```
┌─────────────────────────────────────────────────────────┐
│           THE CRYPTO FOUNDER'S DILEMMA                  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   SHORT-TERM PATH                 LONG-TERM PATH       │
│   ───────────────                 ──────────────       │
│                                                         │
│   ┌─────────────┐                 ┌─────────────┐      │
│   │   Trading   │                 │   Product   │      │
│   │  Platform   │                 │ Development │      │
│   └─────────────┘                 └─────────────┘      │
│         │                               │               │
│         ▼                               ▼               │
│   Quick PMF                       Sustainable Moat     │
│   High Competition                Lower Competition    │
│   Low Defensibility               High Defensibility   │
│   Immediate Revenue               Deferred Revenue     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### The Problem

Crypto's unique dynamics around tokens and speculation push founders toward immediate gratification rather than sustainable business models.

### Key Insight

> "Trading serves an important market function but shouldn't be the final destination. Founders prioritizing genuine product development may ultimately build more valuable enterprises."

## Note 2: Regulations Will Reduce Market Distortions

**Author:** Miles Jennings, a16z crypto Policy Team & General Counsel

### Current State: Legal Uncertainty

Legal uncertainty has forced founders into company-focused regulatory frameworks unsuitable for blockchain networks.

### The Distortions

| Distortion | Effect |
|------------|--------|
| Transparency Avoidance | Founders hide legitimate activities |
| Token Arbitrariness | Legal classification becomes random |
| Performative Governance | Governance theater, not function |
| Legal Optimization | Organizations prioritize protection over purpose |

### The Coming Clarity

Crypto market structure regulation—closer to passage than ever—could reverse these distortions.

```
REGULATORY EVOLUTION
════════════════════════════════════════════════════════

    CURRENT STATE              FUTURE STATE
    ─────────────              ────────────

    • Ambiguity          ──▶   • Clear standards
    • Defensive posture  ──▶   • Transparent operations
    • Token uncertainty  ──▶   • Defined classifications
    • Compliance theater ──▶   • Genuine compliance

════════════════════════════════════════════════════════
```

### Expected Benefits

Following stablecoin legislation successes, market structure regulation would enable networks to operate authentically as:

- [x] Open
- [x] Autonomous
- [x] Composable
- [x] Credibly neutral
- [x] Decentralized
