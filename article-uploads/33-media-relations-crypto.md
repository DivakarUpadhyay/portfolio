## The Core Framework

Three essential questions for communications strategy:

1. **What is your business goal?**
2. **Which audiences must you reach?**
3. **What tactics best reach those audiences?**

> "Your core narrative and messages should hold true no matter who you're talking to or through which medium you're communicating."

## Why Press Matters

Despite skepticism from some tech founders, traditional media relations remains valuable.

### Benefits

| Benefit | Description |
|---------|-------------|
| **Third-party validation** | External credibility |
| **New audience reach** | Beyond "chronically online" |
| **Credibility building** | Legitimacy signals |
| **Crisis preparation** | Pre-built relationships |

### Case Study: Kalshi on CBS

Kalshi's CBS appearance reached broader demographics beyond crypto-native audiences, demonstrating the power of traditional media for market expansion.

## Crisis Prevention

> "The best defense is a good offense."

Building media relationships before crises occur makes difficult conversations easier.

### Crisis Communication Checklist

- [x] Establish reporter relationships proactively
- [x] Prepare key messages in advance
- [x] Identify spokesperson(s)
- [x] Create rapid response protocols
- [ ] Practice crisis scenarios (recommended)
- [ ] Develop holding statements (recommended)

## Tactical Recommendations

### For Early-Stage Startups

1. Focus on owned content first
2. Build community foundation
3. Founders should be active on social
4. Media outreach when newsworthy

### For Growth-Stage Companies

1. Dedicated comms resources
2. Proactive media relationships
3. Regular speaking opportunities
4. Systematic content production
